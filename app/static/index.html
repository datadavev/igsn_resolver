IGSN Resolver

About

This service resolves IGSN identifiers to their target resource or provides metadata about the resource as provided by the DataCite API.

The Open API description for this service is described at /docs.

The IGSN value may be entered in various ways, and it should be noted that IGSN values are not case-sensitive. The following provides representation examples for the same IGSN identifier:

  • IEBJS000K
  • IGSN:IEBJS000K
  • 10.58052/IEBJS000K
  • IGSN:10.58052/IEBJS000K
  • DOI:10.58052/IEBJS000K

Expressed in a URL that resolves to the target: https://igsn.rslv.xyz/igsn:10.58052/IEBJS000K

Expressed in a URL that resolves to the metadata: https://igsn.rslv.xyz/.info/igsn:10.58052/IEBJS000K
